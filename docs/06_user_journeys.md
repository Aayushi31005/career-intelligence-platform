# User Journeys

This document outlines how different users move through the platform to achieve clarity about their career direction. The journeys are designed to be calm, progressive, and non-overwhelming, reflecting the platform’s role as a decision-support system rather than an action-forcing tool.

Each journey focuses on helping users understand *where they are*, *what matters next*, and *how to move forward intentionally*.

---

## Journey 1: Initial Exploration and Orientation

**User Type:** Student / Fresher / Experienced Professional  
**Goal:** Understand what the platform offers and establish a starting point.

### Flow
1. User lands on the platform homepage.
2. The platform briefly explains its purpose: career clarity, readiness, and guidance.
3. User selects their current career stage and primary goal (e.g., preparing for a role, switching roles, improving readiness).
4. User is guided to upload a resume or manually enter skills.

### Outcome
- User understands what the platform does and does not promise.
- Initial profile is created as a foundation for further insights.
- User feels oriented rather than overwhelmed.

---

## Journey 2: Skill Gap Discovery

**User Type:** Student / Fresher  
**Goal:** Identify which skills to prioritize for a target role.

### Flow
1. User selects a target role (and optionally target companies).
2. The system analyzes curated job market data for the selected role.
3. The platform compares required skills with the user’s current skill set.
4. Results are presented as:
   - existing skills
   - missing skills
   - prioritized learning areas

### Outcome
- User gains clarity on high-impact skills.
- Learning effort feels focused rather than scattered.
- Skill gaps are framed as achievable next steps.

---

## Journey 3: Resume Readiness and Alignment

**User Type:** Fresher / Experienced Professional  
**Goal:** Understand how well the resume aligns with a target role.

### Flow
1. User uploads a resume and selects a job description or role.
2. The system performs a heuristic resume alignment analysis.
3. Results are displayed with:
   - alignment indicators
   - missing or underrepresented skills
   - clear explanations of why they matter

### Outcome
- User understands resume strengths and weaknesses.
- Feedback feels transparent and constructive.
- User knows exactly what to modify and why.

---

## Journey 4: Learning Path Guidance

**User Type:** All Personas  
**Goal:** Translate skill gaps into actionable learning steps.

### Flow
1. From the skill gap view, the user explores recommended learning resources.
2. Resources are grouped by skill and difficulty level.
3. User selects resources aligned with their time availability and experience.

### Outcome
- Learning feels purposeful and role-aligned.
- User avoids overconsumption of generic content.
- Clear connection between learning effort and career goals.

---

## Journey 5: Networking Path Exploration

**User Type:** Fresher / Experienced Professional  
**Goal:** Network intentionally toward a target role or company.

### Flow
1. User selects a target company or role.
2. The platform models professional connections as a graph.
3. The system identifies low-friction networking paths (e.g., alumni, mutual connections).
4. Suggested paths are presented with context, not scripts.

### Outcome
- Networking feels structured and less intimidating.
- User understands *why* a connection is relevant.
- Outreach becomes intentional rather than random.

---

## Journey 6: Role Fit and Career Direction

**User Type:** All Personas  
**Goal:** Understand proximity to different roles.

### Flow
1. The platform evaluates user skills and experience against multiple roles.
2. Roles are categorized as:
   - strong fit
   - near fit
   - long-term targets
3. Each category includes an explanation of what differentiates them.

### Outcome
- User gains realistic expectations.
- Career planning becomes incremental rather than binary.
- Motivation is maintained through visible progress.

---

## Design Considerations Across All Journeys

- Users are never forced into a single path.
- Insights are revealed progressively.
- Language remains supportive and non-judgmental.
- The system remains useful even if some advanced features are unavailable.

---

## Closing Note

These journeys emphasize clarity, emotional safety, and forward momentum. They reflect the platform’s commitment to guiding users through career decisions thoughtfully, without pressure or false guarantees.
